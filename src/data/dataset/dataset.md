* trading-data-{commodity}.csv: 2013년부터 최근까지
* news-summary-{commodity}.csv: 2017년 말부터 최근까지 

# 데이터 내용
* news-summary-{commodity}.csv에서 각 열의 의미: {start_date}에서 {end_data}까지 6개월 간의 해당 {commdity} 관련 뉴스 요약
* news-summary-{commodity}.csv에서 "Summary"항목 내의 "{Short/Medium/Long}-term:" 의미: 각각 30일, 90일, 180일 예측을 위한 뉴스 요약
