# 데이터 설명 (Data Description)

## 개요

SentiCast 시스템은 세 가지 광물(금, 은, 구리)의 가격 예측을 위해 두 종류의 데이터를 사용합니다.

## 1. 광물 가격 시계열 데이터 (`prices.csv`)

### 형식

| 컬럼명       | 타입   | 설명                         | 단위      |
|------------|--------|------------------------------|-----------|
| `date`     | string | 날짜 (YYYY-MM-DD)            | -         |
| `gold`     | float  | 금 종가                      | USD/troy oz |
| `silver`   | float  | 은 종가                      | USD/troy oz |
| `copper`   | float  | 구리 종가                    | USD/metric ton |

### 특성

- **금 (Gold)**
  - 안전자산 (Safe-haven asset)
  - 인플레이션 헤지 수단
  - 낮은 변동성, 장기 상승 추세
  - 가격 범위: ~1,500–2,500 USD/troy oz

- **은 (Silver)**
  - 귀금속 + 산업재 이중 특성
  - 금과 높은 상관관계 (귀금속 특성)
  - 태양광 패널, 전기차, 반도체 수요에 민감 (산업재 특성)
  - 금보다 변동성이 큼 (Gold-Silver Ratio 관찰 필요)
  - 가격 범위: ~15–35 USD/troy oz

- **구리 (Copper)**
  - 순수 산업재 (경기 선행지표)
  - 건설, 전기, 전자, 운송 분야 수요에 민감
  - 글로벌 경기 동향 반영
  - 가격 범위: ~6,000–10,000 USD/metric ton

### 상호 연관성

```
금 ↔ 은: 높은 양의 상관관계 (둘 다 귀금속)
은 ↔ 구리: 중간 양의 상관관계 (둘 다 산업재)
금 ↔ 구리: 낮은 상관관계 (금=안전자산, 구리=경기민감)
```

## 2. 뉴스 요약 데이터 (`news.csv`)

### 형식

| 컬럼명             | 타입   | 설명                                              |
|------------------|--------|---------------------------------------------------|
| `date`           | string | 기준 날짜 (YYYY-MM-DD)                            |
| `mineral`        | string | 광물명 (`gold`, `silver`, `copper`, `global`)     |
| `summary_short`  | string | 단기 뉴스 요약 (최근 1–7일, 단기 이슈)            |
| `summary_medium` | string | 중기 뉴스 요약 (최근 1–4주, 중기 트렌드)          |
| `summary_long`   | string | 장기 뉴스 요약 (최근 1–6개월, 장기 구조적 변화)   |

### 뉴스 유형별 특성

- **단기 요약 (short)**: 즉각적인 가격 반응 유발 이벤트
  - 연준 금리 결정, 지정학적 긴장, 공급망 차질, 대형 거래 등
- **중기 요약 (medium)**: 방향성 트렌드
  - 중앙은행 정책, 산업 생산 지표, 무역 데이터, 계절적 수요 패턴
- **장기 요약 (long)**: 구조적 변화
  - 에너지 전환 정책, 지정학적 재편, 기술 혁신, 공급원 다변화

## 3. 입력/출력 형식

### 입력 (기준 날짜 T 기준)

```
time_series:  [T-179 : T]  → shape (180, 3)  # 180일 × 3개 광물
news_short:   [T-6 : T]   → shape (7, 3×1)  # 각 날짜의 단기 요약 임베딩
news_medium:  [T-27 : T]  → shape (4, 3×1)  # 주별 중기 요약 임베딩
news_long:    [T-5 : T]   → shape (6, 3×1)  # 월별 장기 요약 임베딩
```

### 출력

```
predictions: 30일, 90일, 180일 후 가격 → shape (3, 3)  # 3개 지평 × 3개 광물
intervals:   예측 신뢰구간 (90% CI)     → shape (3, 3, 2)  # lower/upper bounds
```

## 4. 데이터 전처리

- **정규화**: 각 광물별 독립적으로 min-max 또는 z-score 정규화 적용
  - 광물별 가격 범위 차이를 보정 (금 ~2000, 은 ~25, 구리 ~8000)
  - 학습/검증/시험 분리 후 학습 데이터 통계만 사용

- **결측치 처리**: 선형 보간 (주말/공휴일 등)

- **이상치 처리**: 3σ 기준 클리핑 후 보간

## 5. 데이터 분리 (Rolling Forward)

```
전체 기간: 2019-01-01 ~ 2024-12-31 (예시)

학습: 초기 60% 구간
검증: 중간 20% 구간 (학습 이후)
시험: 최종 20% 구간 (검증 이후)

롤링 포워드: step=30일 단위로 윈도를 앞으로 이동하며 평가
```

## 6. 샘플 데이터 생성

실제 데이터가 없는 경우 `data/sample/` 디렉토리의 샘플 데이터를 사용할 수 있습니다:

```bash
python src/data/generate_sample.py
```

이 스크립트는 현실적인 가격 패턴(추세, 계절성, 노이즈)과 임의 뉴스 임베딩을 생성합니다.
